# Сервис автомазизации операций составления приложений к дипломам

[пгту-ртф.рф](https://xn----etb9agicel.xn--p1ai/)

Проект предназначен для генерации сводных таблиц на основе данных, а также создания XML-документов. Он состоит из нескольких модулей:

- `java-api` – Java-сервис, который предоставляет HTTP-интерфейс и служит фронтом для операций.
- `python-engine` – Python-движок для анализа и подсветки проблемных ячеек при создании сводных таблиц.
- `python-xml-engine` – Python-модуль, отвечающий за генерацию XML-документов.

## Запуск

Работа всех компонентов организована через Docker. Соберите и запустите контейнеры командой:

```sh
docker-compose up --build
```

Образы складываются из соответствующих `Dockerfile` в каталогах `java-api`, `python-engine` и `python-xml-engine`.

Сервисы взаимодействуют внутри общей сети, порты и переменные настраиваются в `docker-compose.yml`.

Внутри контейнеров:

- `java-api` — Spring Boot приложение, обрабатывающее HTTP-запросы и перенаправляющее их в движки.
- `python-engine` — прослушивает API для создания сводных таблиц.
- `python-xml-engine` — отвечает за генерацию XML-документов.

## Функциональные разделы

### "Создать сводную таблицу"

1. Пользователь загружает исходный Excel-файл (или передает данные по API).
2. `java-api` пересылает задачу в `python-engine`, где выполняется анализ, подсветка проблемных ячеек и формирование итоговой сводной таблицы.
3. Особенности:
   - В движке происходит проверка и выделение ячеек с неконсистентными данными;
   - Результат возвращается в виде нового Excel-файла с выделениями и/или JSON, в зависимости от API.
   - Параметры конфигурации (например, критерии объединения) задаются через `Config` в Java и `settings` в Python.

### "Создать документ XML"

1. После получения данных для XML, `java-api` направляет запрос в `python-xml-engine`.
2. Этот движок строит XML на основе шаблона и данных, применяя дополнительные правила.
3. Особенности:
   - Генерация проходит в `app/xml_generator.py`, где содержатся нюансы формирования тегов и атрибутов;
   - В конфигурации можно менять структуру выходного документа;
   - Если данные некорректны, возвращается ошибка с описанием проблемных полей.

## Использование в Docker

Для контейнеризации доступны Dockerfile в каждом модуле. `docker-compose.yml` связывает сервисы между собой, устанавливает сети и порты.
