🧮 Создать сводную таблицу

Сервис поможет создать сводную таблицу результатов обучения студентов по всем дисциплинам учебного плана (в формате XLSX). Столбцы — это дисциплины, строки — оценки студентов.

📋 Обратите внимание: • Оба файла должны быть в формате XLSX.
• Список дисциплин формируется на основе учебного плана.
• Названия практик (Учебная, Производственная, ...) лучше взять из сводного документа успеваемости студентов, а уже в созданной сводной таблице заменить на соответствующие названия из учебного плана.
• Раздел курсовых работ заполнится автоматически из сводного документа успеваемости студентов.
• Пример сводного документа успеваемости студентов — нажмите здесь (можно выгрузить через деканат).
• Пример списка дисциплин — нажмите здесь (цветом выделены обязательные заголовки).

📄 Создать документ XML

Сервис поможет конвертировать проверенную сводную таблицу в документ XML для программы КиберДИПЛОМ.

Важно! После импорта сгенерированного документа XML в самой программе КиберДиплом нужно:
1. Добавить во вкладке «Госаттестация» поле «Государственный экзамен» и соответственно саму оценку.
2. Добавить руководителя (ФИО ректора).
3. Проверить, что все данные корректно отобразились в программе, особенно оценки, названия дисциплин и практик.
