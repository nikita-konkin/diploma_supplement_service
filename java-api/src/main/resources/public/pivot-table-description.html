Руководство по работе со сводной таблицей

Описание таблицы

Сводная таблица "pivot_report" содержит результаты обучения студентов по всем дисциплинам учебного плана. Таблица включает следующую информацию:

  • Столбец "Дисциплины": содержит названия учебных дисциплин в формате: Название_тип_зачетные_единицы (например, "Математика_дисциплина_5")
  • Столбцы с именами студентов: содержат оценки по соответствующим дисциплинам
  • Разделы таблицы: дисциплина, факультатив, практика, госэкзамен, курсовая

Формат оценок

В ячейках с оценками используются следующие обозначения:

  • 3, 4, 5: числовые оценки за экзамены
  • 6: зачет (зачтено)
  • ? и !: пропуск или отсутствие оценки
  • Пустая ячейка: оценки нет в базе данных
💡 Зачетные единицы
Зачетные единицы дисциплины указаны в ее названии после знака подчеркивания "_". Например, в "Математика_дисциплина_5" зачетных единиц: 5.

Инструкции по редактированию таблицы

⚠️ ЧТО НЕОБХОДИМО СДЕЛАТЬ:

1. Убрать лишние строки

Удалите все строки в конце таблицы с названиями типа "Элективная дисциплина 1 (...)", "Элективная дисциплина 2 (...)" и т.д.

  • Эти строки содержат полные названия элективных дисциплин и должны быть удалены

2. Проверить корректность данных

  • Зачетные единицы: убедитесь, что зачетные единицы в названиях дисциплин соответствуют учебному плану
  • Элективные дисциплины: проверьте правильность выбора элективных дисциплин студентами, уберите "(Элективная дисциплина 1)" и т.д. из названий дисциплин, оставив только название и тип (например, "Иностранный язык_дисциплина_3")
  • Названия практик: в скобках должен быть указан тип практики (например, "технологическая (проектно-технологическая)" или "преддипломная практика"), например, "Учебная практика (ознакомительная)_практика_6"
  • Названия курсовых работ и проектов: в скобках должен быть указан тип курсовой (например, "курсовая работа" или "курсовой проект"), например, "Цифровые устройства и микропроцессоры (курсовая работа)_курсовая_3"
  • Оценки: проверьте корректность оценок.

3. Сохранить структуру таблицы

❗ ВАЖНО!
После всех исправлений обязательно сохраните заголовки разделов:
  • Дисциплины
  • факультатив
  • практика
  • госэкзамен
  • курсовая

Сохраняйте типы записи в названиях (дисциплина, факультатив и т.п) (например, "Математика_дисциплина_5", "Учебная практика (ознакомительная)_практика_6" и т.д.).

✅ Рекомендации после редактирования:
• Проверьте, что все обязательные дисциплины имеют оценки
• Убедитесь, что сумма зачетных единиц соответствует требованиям программы
• Сохраните отредактированный файл под новым именем
